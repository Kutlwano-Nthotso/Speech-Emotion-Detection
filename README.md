# 🎤 Speech Emotion Recognition System

## 📌 Overview
A Speech Emotion Recognition system that identifies human emotions from audio recordings. It uses MFCC feature extraction and a Random Forest classifier, achieving 88% accuracy. The system supports both real-time microphone input and file-based prediction.

---

## ⚙️ Features
- 🎧 Detects emotions from speech audio
- 🧠 MFCC feature extraction
- 🌲 Random Forest classifier
- 📊 Achieves ~88% accuracy
- 🎙️ Real-time microphone prediction
- 📁 File-based audio prediction

---

## 🧠 Emotions Detected
- Neutral
- Calm
- Happy
- Sad
- Angry
- Fear
- Disgust
- Surprise
